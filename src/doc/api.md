###主页
> **接口1**：歌手排行榜前四名 (返回一个有四个元素的数组)       
> 歌手id                          
> 歌手名                          
> 歌曲总数                          
> 歌手图片                          
```
[
  {
    id: '23486846328',
    name: '薛之谦',
    count: '188',
    image: 'http://foo.com/image/singer.png'
  }
  ...排行前四个歌手
]
```

> **接口2**: 任选四个歌曲排行榜，返回四个排行榜每个的前四首歌曲。          
> 榜单标题   
> 榜单描述                  
> 歌曲: (歌曲id，歌曲名，歌手名)
 
 
```
[
  {
    title: '巅峰榜',
    des: '描述',
    songs: [
      {
        id: '歌曲id',
        name: '歌曲名',
        artist: '歌手名'
      }
      ...前四首歌曲
    ]
  }
  ...共四个排行榜
]
```


###歌手
